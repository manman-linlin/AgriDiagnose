# 🌾 AgriDiagnose — 农作物病虫害智能诊断系统

基于深度学习的农作物叶片图像识别系统，结合大模型 Agent 生成防治建议。上传一张叶片照片，即可自动识别病害类别并获取结构化防治方案。

> 🏫 大学三年级小学期课程设计项目 | 软件开发 + 深度学习

## ✨ 功能特性

- **🔍 智能识别** — 上传农作物叶片图片，ConvNeXt-Tiny 模型自动分类 38 种病害/健康状态，准确率 **99.80%**
- **🌐 Web 交互** — Vue3 单页应用，支持拖拽上传、置信度可视化、Top-3 预测排名
- **📋 历史记录** — 自动保存每次诊断，支持缩略图浏览和删除管理
- **📊 数据统计** — ECharts 饼图实时展示各类病害诊断分布
- **🇨🇳 中文输出** — 中英文类别映射，诊断结果直接显示中文病害名
- **🧠 Agent 模块** — 预留大模型接口，可接入 LLM 生成防治措施、风险提示等结构化建议

## 🚀 快速开始

### 1. 环境准备

```bash
# 安装模型依赖
cd 2.Model
pip install -r requirements.txt

# 安装 Web 依赖
cd ../3.Web/Backend
pip install -r requirements.txt
```

### 2. 训练模型（或使用预训练权重）

```bash
cd 2.Model
python train.py
```

> 已提供训练好的权重 `2.Model/Weights/convnext_tiny_best.pth`（111.5 MB），可直接使用。

### 3. 启动 Web 服务

```bash
cd 3.Web/Backend
python run.py
```

浏览器访问 **http://localhost:8000** 即可使用。
