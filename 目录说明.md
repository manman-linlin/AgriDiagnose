# AgriDiagnose — 项目目录说明

## 项目简介

农作物病虫害智能诊断系统。上传叶片图片 → 深度学习模型识别病害 → 大模型 Agent 生成防治建议。

## 目录结构

```text
AgriDiagnose/
├─ 1.Docs/        文档（详细设计文档）
├─ 2.Model/       图像识别模型（训练/推理/权重/数据）
├─ 3.Web/         Web 系统（FastAPI 后端 + Vue3 前端）
├─ 4.Agent/       大模型 Agent 模块（防治方案生成）
├─ 5.Test/        系统测试
└─ README.md      项目主页
```

## 各目录说明

### 1. Docs — 项目文档
- `详细设计文档.md`：系统背景、需求分析、架构设计、模块划分、接口设计、测试方案

### 2. Model — 图像识别模型
- `config.py`：训练与推理配置
- `data.py`：数据集加载、增强、train/val 划分
- `model.py`：ConvNeXt-Tiny 模型构建（timm，ImageNet 预训练）
- `train.py`：训练入口，每轮输出整体准确率 + 逐类准确率
- `infer.py`：单张图片推理（命令行）
- `utils.py`：工具函数（种子、指标、checkpoint 读写、中文映射）
- `Data/`：PlantVillage 数据集 (54,305 张, 38 类) + class_labels 中英对照
- `Weights/`：训练权重 convnext_tiny_best.pth (99.80% 准确率)
- `scripts/`：数据下载、划分等工具脚本

### 3. Web — Web 应用系统
- `Backend/`：FastAPI 后端
  - `app/main.py`：API 路由（诊断/历史/统计）
  - `app/services/classifier.py`：模型推理服务（常驻内存）
  - `run.py`：一键启动脚本
- `Frontend/`：Vue3 前端
  - `index.html`：诊断/历史/统计三页签 SPA

### 4. Agent — 大模型 Agent
- 预留模块，用于调用大模型 API 生成病虫害防治建议

### 5. Test — 测试
- 系统测试用例、测试报告、运行截图

## 技术栈

| 层级 | 技术 |
|------|------|
| 模型 | PyTorch + timm (ConvNeXt-Tiny) |
| 后端 | Python FastAPI |
| 前端 | Vue 3 + ECharts |
| 数据 | PlantVillage (38 类) |

## 快速启动

```bash
cd 3.Web/Backend
pip install -r requirements.txt
python run.py
# → http://localhost:8000
```
